## 4. Convolutional Neural Networks

### week1.卷积神经网络
1.1 卷积(kernel/filter)
- 图像维度太多，传统DNN参数太多，容易过拟合
- 用卷积核跟图像进行卷积
- 通过调整卷积核的值，可以实现边缘检测的功能
- 卷积核的值可以通过神经网络学习得到
- 卷积核维度通常是奇数

1.2 padding
- 卷积核维度小的化，图像经过卷积后会越来越小，同时边缘的点被卷积的次数太少
- padding：沿着图像边缘，填充像素
- 两类convolutions：
    - Valid：不做padding
    - Same：Pad so that output size is the same as the input size

1.3 卷积步长(Stride)
- 卷积核移动的步长
